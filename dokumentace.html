Dokumentace Nexiflow

Vítejte v dokumentaci. Chatbot na vašem webu máte za 5 minut — bez programování.

Rychlý start

1

Vytvořte účet

Přejděte na dashboard.nexiflow.cz/register a zaregistrujte se zdarma. Automaticky získáte 14 dní Business plánu.

2

Vytvořte chatbota

V dashboardu klikněte na Chatboti → Nový chatbot. Pojmenujte ho, nastavte barvu a uvítací zprávu.

3

Přidejte znalosti

V záložce Znalostní báze vložte texty o vašem produktu, FAQ nebo importujte články. Čím víc dat, tím přesnější odpovědi.

4

Vložte na web

Zkopírujte embed kód (viz níže) a vložte ho na váš web. Hotovo.

Vytvoření chatbotu

Každý chatbot má tyto základní nastavení:

  • Jméno bota – zobrazuje se zákazníkům v hlavičce widgetu
  • Systémová instrukce – definuje charakter a chování bota (např. „Jsi zákaznická podpora e-shopu XY, odpovídáš česky, jsi přátelský")
  • Uvítací zpráva – první zpráva zobrazená při otevření chatu
  • Barva widgetu – přizpůsobí se vašemu brandu
  • Avatar bota – vlastní obrázek nebo výchozí ikona
Tip: Systémová instrukce je nejdůležitější nastavení. Čím konkrétnější instrukci napíšete, tím lépe bude bot odpovídat ve vašem stylu.

Znalostní báze

Znalostní báze je základ kvalitních odpovědí. Můžete přidat:

  • Textové články – FAQ, popis produktů, ceník, podmínky
  • Přílohy – PDF dokumenty (plán Solo a výše)
  • URL import – Nexiflow stránku automaticky zpracuje (připravujeme)

Doporučujeme začít s 5–10 klíčovými FAQ, které zákazníci nejčastěji pokládají.

HTML / jakýkoliv web

Vložte tento kód před uzavírací tag </body> na každé stránce, kde chcete chatbot zobrazit:

<!-- Nexiflow chatbot -->
<script
  src="https://dashboard.nexiflow.cz/api/widget/VAS_BOT_ID"
  async
></script>

VAS_BOT_ID najdete v dashboardu pod Chatboti → Váš bot → Embed kód.

Bot ID je unikátní pro každý chatbot. Pokud máte více chatbotů, každý má jiné ID.

WordPress plugin

Pro WordPress máme hotový plugin – instalace je jednoduchá:

1

Stáhněte plugin

V dashboardu přejděte na Chatboti → Embed kód → WordPress a stáhněte ZIP soubor.

2

Nainstalujte do WordPressu

WordPress admin → Pluginy → Přidat nový → Nahrát plugin → vyberte ZIP → Aktivovat.

3

Vložte Bot ID

V nastavení pluginu vložte váš Bot ID z dashboardu. Plugin automaticky přidá chatbot na celý web.

Webflow

V Webflow přejděte na Projekt nastavení → Custom Code → Footer Code a vložte embed script (stejný jako pro HTML web).

Inbox & live chat

Veškeré konverzace zákazníků najdete v Inbox sekci dashboardu. Můžete:

  • Přebírat konverzaci od bota ručně (live chat mód)
  • Označovat konverzace štítky a přiřazovat členům týmu
  • Přidávat interní poznámky
  • Exportovat historii do CSV

Webhooky

Nexiflow umí odesílat data o nových konverzacích na váš webhook endpoint (plán Business+). Využití:

  • Propojení s CRM (HubSpot, Pipedrive)
  • Notifikace do Slacku / Teams
  • Automatizace v Make nebo Zapier

Nastavení: Dashboard → Nastavení → Webhooky

Každý request obsahuje hlavičku X-Nexiflow-Signature pro ověření pravosti.

API přístup (Enterprise)
